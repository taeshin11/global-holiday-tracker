2. Description of Service

Global Holiday Tracker is a free, web-based information service that aggregates and displays public holiday data for countries worldwide. The Service provides the following core features: a multi-country holiday list with countdown timers; an interactive monthly calendar with color-coded holiday markers; tracking of major e-commerce and commercial shopping events; .ics calendar file export functionality compatible with standard calendar applications; a language switcher supporting eight languages; and a Progressive Web App (PWA) experience for mobile device installation.
